CSS INTERVIEW QUESTIONS AND ANSWERS (DAY 2)

  1. What is box sizing?

    Defines how the width and height of an element are calculated

    The box-sizing property allows us to include the padding and border in an element's total width and height

  2. What is animation delay?

    The animation-delay CSS property specifies the amount of time to wait from applying the animation to an element before beginning to perform the animation.

  3. Which property you will use to merge cells vertically in grid?

    CSS grid-column property.

  4. Which property you will use to merge cells horizontally in grid?

    CSS grid-row property.

  5. what is the difference between display none and visibility hidden?

    visibility:hidden hides the element, but it still takes up space in the layout.

    display:none removes the element from the document. It does not take up any space.
